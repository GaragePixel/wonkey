(Mx2cc v1.1.15 • Ted2go v2.13)

Welcome to wonkey, an free and open source, cross platform programming language by Wonkey-Coders.

Some example programs can be found in the examples directory.

Wonkey is a absolutly free project fully backed by iDkP from GaragePixel. If you like what you see, please consider contributing to development on the Github repository: https://github.com/wonkey-coders/wonkey.

For Wonkey discussion and programming help, feel free to drop by the discord (invitation only for now).

Please post any bug reports or feature requests to the Wonkey github repository: https://github.com/wonkey-coders/wonkey.
